# Улучшения обработки ошибок и UX загрузки

## 🔧 Сделано

### 1. **Расширенная обработка ошибок в `lib/api.ts`**
   - Все функции теперь выбрасывают читаемые сообщения об ошибках
   - Проверка размера файла (макс 5MB)
   - Проверка типа файла при загрузке
   - Детальные сообщения для каждого типа ошибки

### 2. **Улучшенный UX при загрузке файлов в `OrderForm.tsx`**
   - Проверка интернета перед загрузкой (`!navigator.onLine`)
   - Отдельный обработчик ошибок для каждого файла
   - Отображение названия файла при ошибке
   - Разные сообщения для сетевых ошибок и ошибок загрузки

### 3. **Расширенная обработка в `use-orders.ts`**
   - Добавлен state `error` для отслеживания ошибок
   - Слушатель события `online` для автоматической перезагрузки при восстановлении интернета
   - Проверка `navigator.onLine` при всех операциях
   - Разные сообщения для сетевых ошибок и ошибок БД

### 4. **Скелетоны для загрузки**
   - `OrderCardSkeleton.tsx` - скелетон карточки заказов
   - `OrderFormSkeleton.tsx` - скелетон формы заказов
   - Отображение 8 скелетонов во время загрузки

### 5. **Визуализация статуса загрузки в `page.tsx`**
   - Показ скелетонов пока `isLoading === true`
   - Отображение ошибки с Alert компонентом если `error !== null`
   - Обновлена Dashboard для получения `error` из `useOrders`

### 6. **Мониторинг подключения к интернету**
   - Компонент `ConnectionStatus.tsx` отображает статус интернета
   - Слушатели `online`/`offline` событий в главном компоненте
   - Автоматическая перезагрузка данных при восстановлении

## 🎯 Улучшения UX

1. **Видимость процесса**: Пользователь видит скелетоны вместо пустой страницы
2. **Информативность**: Разные сообщения об ошибках для разных ситуаций
3. **Автоматическое восстановление**: Данные автоматически загружаются при восстановлении интернета
4. **Статус подключения**: Видимый Alert показывает отсутствие интернета
5. **Валидация файлов**: Проверка размера и типа перед загрузкой

## 🔗 Типы ошибок

- **Сетевые**: "Нет интернета" → предложение проверить соединение
- **БД ошибки**: Специфическое сообщение об ошибке Supabase
- **Файлы**: Минимум 5MB → "Размер файла превышает лимит"
- **Форма**: Валидация полей как раньше
