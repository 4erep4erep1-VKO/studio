# MontazhkaPRO v 3.0

Профессиональная система управления рекламными монтажами.

## Особенности
- **Облачная база данных**: Все основные данные безопасно хранятся в Supabase.
- **Роли**: Разделение интерфейса для Администратора и Монтажника.
- **Общие заказы**: Возможность создавать задачи, которые монтажники забирают самостоятельно.
- **Мультимедиа**: Поддержка прикрепления нескольких фото и вставки скриншотов через Ctrl+V.
- **Уведомления**: Система оповещений со звуковым сопровождением.
- **Система комментариев**: Общение между пользователями в реальном времени через Supabase.
- **Админ-панель**: Отдельный интерфейс для администраторов с статистикой и расширенной фильтрацией.
- **Realtime синхронизация**: Автоматическое обновление данных в реальном времени.
- **Mobile First**: Полная адаптация под мобильные браузеры.

## Как опубликовать (Deploy)

Рекомендуется использовать Vercel для простого и быстрого деплоя Next.js приложения:

1. Подключите ваш репозиторий на GitHub к [Vercel](https://vercel.com/).
2. Vercel автоматически создаст Production среду и будет деплоить при каждом push.
3. Добавьте переменные окружения в настройках проекта:
   - `NEXT_PUBLIC_SUPABASE_URL`
   - `NEXT_PUBLIC_SUPABASE_ANON_KEY`
4. Приложение будет опубликовано на бесплатном домене `*.vercel.app`.

## Настройка Supabase

1. Создайте проект в [Supabase](https://supabase.com/).
2. Получите URL проекта и анонимный ключ.
3. Скопируйте `.env.example` в `.env.local` и заполните переменные:
   ```bash
   cp .env.example .env.local
   ```
4. Добавьте переменные окружения в `.env.local`:
   ```
   NEXT_PUBLIC_SUPABASE_URL=your_supabase_project_url
   NEXT_PUBLIC_SUPABASE_ANON_KEY=your_supabase_anon_key
   ```
5. Выполните SQL-скрипт `supabase-comments.sql` в SQL-редакторе Supabase для создания таблицы комментариев.
6. Убедитесь, что Realtime включен для таблицы `comments`.

## Админ-панель

Администраторы имеют доступ к специальной панели управления (`/admin`) с расширенными возможностями:

### Статистика в реальном времени
- **Всего заказов**: Общее количество заказов в системе
- **В работе**: Количество заказов в статусе "В работе"
- **Выполнено за сегодня**: Заказы, завершенные сегодня

### Расширенная фильтрация
- **По статусу**: Все, В работе, Завершен, Отклонен
- **По исполнителю**: Фильтр по конкретному монтажнику или общие заказы
- **Поиск**: По названию объекта

### Функциональность
- Просмотр всех заказов независимо от назначения
- Изменение статуса заказов
- Доступ к комментариям в реальном времени
- Управление исполнителями заказов

## Система комментариев

- **Realtime обновления**: Комментарии появляются мгновенно для всех пользователей.
- **Пользовательский интерфейс**: Комфортный чат в каждой карточке заказа.
- **Безопасность**: RLS политики обеспечивают доступ только аутентифицированным пользователям.
