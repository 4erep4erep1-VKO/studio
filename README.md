# MontazhkaPRO v 3.0

Профессиональная система управления рекламными монтажами.

## Ключевые особенности

- **Разделение ролей**:
  - **Администратор**: Полный доступ к созданию, редактированию и управлению заказами и пользователями.
  - **Монтажник**: Просмотр назначенных заказов, обновление статусов и загрузка отчетных файлов.

- **Единый пул заказов с Real-time**:
  - Все заказы находятся в общем списке.
  - Изменения, сделанные одним пользователем (например, создание заказа администратором или смена статуса монтажником), **мгновенно** отображаются у всех остальных подключенных пользователей без необходимости обновлять страницу.
  - Архитектура построена на **Supabase Realtime Subscriptions**, что гарантирует актуальность данных.

- **Надежная обработка данных и UX**:
  - Отображение скелетонов во время загрузки данных для предотвращения "пустого экрана".
  - Четкие сообщения об ошибках (отсутствие интернета, ошибки базы данных, неверный формат файла).
  - Автоматическая перезагрузка данных при восстановлении интернет-соединения.
  - Валидация файлов (размер до 5MB, тип) на стороне клиента перед отправкой на сервер.

## Технологии
- **Frontend**: Next.js 15 (App Router), React, TypeScript
- **UI**: Tailwind CSS, ShadCN UI, Lucide Icons
- **Backend & DB**: Supabase (PostgreSQL, Realtime Database, Storage, Auth)

## Развертывание и запуск

1. Подключите репозиторий на GitHub к [Vercel](https://vercel.com/).
2. Vercel автоматически создаст Production-окружение и будет выполнять деплой при каждом `push` в основную ветку.
3. Добавьте переменные окружения в настройках проекта Vercel:
   - `NEXT_PUBLIC_SUPABASE_URL`
   - `NEXT_PUBLIC_SUPABASE_ANON_KEY`
4. Приложение будет опубликовано на бесплатном домене вида `*.vercel.app`.
